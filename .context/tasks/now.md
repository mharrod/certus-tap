## Task Template

When adding new tasks, use this format:

```markdown
### [Task Name]

**Owner:** [agent-role or human]
**Project/Proposal:** [docs/reference/... or epic link]
**Phase/Milestone:** [phase or sprint]
**Files:** [list of files this task may edit]
**Don't Touch:** [files this task must not edit]
**Status:** [in-progress|blocked|completed]
**Description:** [what needs to be done]
```

Example:

```markdown
### Add API versioning documentation

**Owner:** implementer
**Project/Proposal:** docs/reference/roadmap/proposals/core/service-layer-migration-proposal.md
**Phase/Milestone:** Phase 1 – API polish
**Files:** docs/reference/api/api-versioning.md
**Don't Touch:** docs/framework/architecture/\*
**Status:** in-progress
**Description:** Document the API versioning strategy based on api-versioning-proposal.md
```
